# Portfolio Tracker Live

Real-time portfolio tracking with Firebase sync and live stock price updates.

## What It Is

Portfolio Tracker Live is a web application that lets you build and monitor a live investment portfolio. Configure your holdings and market position, then watch real-time price updates from Finnhub automatically refresh your portfolio value and gains. All portfolio settings are saved to Firebase for seamless access across devices, with a polished dark-themed dashboard displaying performance metrics and position details.

## Tech Stack

- React 19 with TypeScript
- Firebase for authentication and data persistence
- Finnhub API for live stock prices
- MobX for state management
- Tailwind CSS for styling

## Getting Started

```
npm install
npm run dev
```

Create a portfolio by selecting stocks and setting your market position weight. The app syncs your data to Firebase and begins fetching live price updates automatically.
